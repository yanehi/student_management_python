# student_management_python
Simple student management project for practice programming with Python 

## Prerequisites

### Python

* Install python

* Install packages
```bash
pip3 install -r requirements.txt
```

* Start project
```bash
python3 main.py
```

* Run your tests
```bash
python3 -m unittest tests.test_calc -v
```
